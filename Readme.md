# ConDeBot
Un con de bot IRC. 

## Features
* Display his own version  
``!cdb version``
* Make a good coffee  
``!cdb coffee``
* Display actual weather and temperature of a location  
``!cdb weather CITY_NAME``
* Display random quotes and specific quotes from Kaamelott.txt  
``!cdb kaamelott [-q ID]``
* Display help  
``!cdb help``
